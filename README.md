# Word Counter App

Counts number of words across files specified as arguments.

# Requirements

- JDK
- Maven
- Chrome (for Eclipse Che)

### Build the app:

```console
mvn clean install
```

### Example usage: 

```console
$ java -jar target/wordcounter-1.0-SNAPSHOT.jar file1 file2 
hello: 10 
world: 5 
example: 3
```
